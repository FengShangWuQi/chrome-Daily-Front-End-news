
> # Chrome Extension for Daily Front End News

## 使用

- 下载 [chrome-Daily-Front-End-news.crx](https://github.com/FengShangWuQi/chrome-Daily-Front-End-news/blob/master/chrome-Daily-Front-End-news.crx)；
- 将 .crx 文件拖入扩展程序页面即可安装；
- 点击插件图标查看每日资讯；
